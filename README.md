## Session 1 (Day 1): Organic Sculpting in Blender

**Blender** is a free, professional 3D tool. Two key modes:

| Mode | Use |
|------|-----|
| **Object Mode** | Move, rotate, scale whole objects |
| **Edit Mode** | Reshape by moving vertices, edges, faces (press Tab to switch) |

**In-class activity: Low-Poly Pokemon**
Create a simplified (low-poly) Snorlax or Pikachu:
1. Open Blender, delete the default cube (X > Delete)
2. Add a UV Sphere (Shift+A > Mesh > UV Sphere)
3. Enter Edit Mode (Tab) and reshape vertices
4. Export as `.stl` for printing

- Blender download: [blender.org](https://www.blender.org/download/) (free)

## Session 2 (Day 2): Parametric Design in Fusion 360 / Tinkercad

For functional objects, **parametric modeling** uses exact measurements.

**Tinkercad** (browser, beginner-friendly): [tinkercad.com](https://www.tinkercad.com)
- Drag, drop, and resize pre-made shapes
- Use "Hole" shapes to subtract material

**Fusion 360** (advanced, free for education): [autodesk.com/fusion360](https://www.autodesk.com/products/fusion-360/education)

**In-class activity:** Design a custom headset stand with specific dimensions.
