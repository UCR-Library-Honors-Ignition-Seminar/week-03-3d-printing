# Week 3 Assignment: Additive Manufacturing (3D Printing)

**Due:** End of Week 3 | **Late policy:** 20% deduction; accepted through end of Week 4

---

## Project

Design and 3D print an object of your choice.

### Option A: Functional Object
A desk organizer, phone stand, cable holder, bookend, or similar. Must have at least one functional feature.

### Option B: Artistic Object
A low-poly character, sculpture, or decorative piece. Must be based on a recognizable concept.

### Requirements
- The model must be **your own design** (not downloaded from Thingiverse)
- The object must be **physically printed** — document with a photo
- Include the model file (`.stl`, `.blend`, or Tinkercad share link)

---

## Process Documentation (PDF Tutorial)

Create a **PDF tutorial** of your making process:
- At least 4 screenshots of your modeling process
- Short captions explaining each step
- A photo of the final printed object

Export from Google Slides, Canva, or Word as PDF.

---

## Reflection

Write a **~300-word reflection** using the [reflection template](reflection-template.md).

---

## Submission Instructions

1. Create a `week-03/` folder in your GitHub Classroom repo
2. Upload your process PDF and model file (or Tinkercad link in `links.md`)
3. Upload a photo of the printed object
4. Create `week-03/reflection.md`
5. Commit and submit repo URL on Canvas
